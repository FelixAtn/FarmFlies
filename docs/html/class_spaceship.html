Detailed Description

Represents the player's spaceship in the game, managing movement, shooting, and state updates.

The Spaceship class handles the functionality of the spaceship, including firing projectiles, updating its position, checking its alive status, and drawing it to the screen. It also manages the spaceship's projectiles and updates their states.
