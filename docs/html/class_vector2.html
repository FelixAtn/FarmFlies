Detailed Description

template<typename T>
class Vector2< T >

A basic 2D vector class for game development.

This class represents a 2D vector and provides useful methods for mathematical operations. It can be used for various purposes like positioning, movement, and physics calculations in games.

Template Parameters
TThe type of the vector components (e.g., float or int).
